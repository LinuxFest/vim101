What is vim

vim.org
Vim is a highly configurable text editor built to enable efficient text editing. Vim is often called a "programmer's editor," and so useful for programming that many consider it an entire IDE!

Why learn vim

  • You need a commandline editor
  • You want to use more out of your keyboard
  • Use a programming language which doesn't have a good editor
  • You want to customize everything!

Modes in vim

  • Normal Mode : For navigation and manipulation of text
  • Insert Mode : Inserting new text
  • Select Mode : Selecting some text
  • Command mode : For entering editor commands

Normal mode example commands

  • 'dfa' : delete until the next 'a'
  • '5j' : go down 5 lines
  • 'd5j' : delete 5 lines below
  • ':5' : goto line 5

Scenarios

  • Implementing functions in C header file
  • Getting output from command and inserting into file
  • Working with tags in html
  • Doing inline calculation
